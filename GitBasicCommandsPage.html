Batch Commands

"Batch Commands for Windows OS/cmd.exe interpreter"

dir command

dir: list all visible folders and files in a directory

dir -force: list all visible or hidden folders and files in a directory

ver command

ver: Displays current windows version

cls command

cls: clear screen

pause command

pause : wait for user input

Data Types | Variables

To define a variable use set

Example : set varName=10

To use variable use %varName%

set command

set : displays all env. variables

example: >> set :displays all env. Variables

set varName:displays varName value

Example: >> set USER : displays all users

String Concatenation

To concatenate two strings , place them beside each other in one variable

Example: set fullName=Amr

set fullName=%fullName% Abdeen

Note:

if we use %fullName% to display fullName it will only show the first name , rather we should use echo %fullName%

BatchScript.bat

You can bundle all batch commands in one batch file and run it

echo

echo used to display variables and commands

>> echo %varName%

help

help: (alone) used to display all commands with description , or a specific command

>> help CMD

You can also use:

>> CMD /? to display CMD description

@ Symbol

@ Symbol: used to disable displaying commands in a batch file

Example: in a batch file : @echo OFF
At the begining of the batch file will disable/suppress displaying all commands in CMD screen
while running the batch file

% , ^ Escape Operators

1) %

>> echo Hey %%Escape Operator%%
this will display: Hey %Escape Operator%

2) ^

>> echo ^<data^>
this will be displayed as :<data>
