dir command

dir: list all visible folders and files in a directory

dir -force: list all visible or hidden folders and files in a directory

ver command

ver: Displays current windows version

cls command

cls: clear screen

pause command

pause : wait for user input

Data Types | Variables

To define a variable use set

Example : set varName=10

To use variable use %varName%

set command

set : displays all env. variables

example: >> set :displays all env. Variables

set varName:displays varName value

Example: >> set USER : displays all users

String Concatenation

To concatenate two strings , place them beside each other in one variable

Example: set fullName=Amr

set fullName=%fullName% Abdeen

Note:

if we use %fullName% to display fullName it will only show the first name , rather we should use echo %fullName%

BatchScript.bat

You can bundle all batch commands in one batch file and run it

echo

echo used to display variables , commands and messages

>> echo %varName%

help

help: (alone) used to display all commands with description , or a specific command

>> help CMD

You can also use:

>> CMD /? to display CMD description

@ Symbol

@ Symbol: used to disable displaying commands in a batch file

Example: in a batch file : @echo OFF
At the begining of the batch file will disable/suppress displaying all commands in CMD screen
while running the batch file

% , ^ Escape Operators

1) %

>> echo Hey %%Escape Operator%%
this will display: Hey %Escape Operator%

2) ^

>> echo ^<data^>
this will be displayed as :<data>

set /a mod=10%3 this will work fine in CMD
But in script we should write it as set /a mod=10 %% 3

Labels

Label is just giving a name to a group of CMDs

To write a label use :labelName

Any CMD under that label will be considered a part of it

:takeUserInput set /p userInput=Enter your name

Laberls works perfectly with Goto [ Explained in next section ]

goto

goto jumps(Go) to a specific label

Example:

@echo OFF
:Start
echo Start Batch Script ...
goto :displayOption
:takeUserInput
set /p userInput=Enter your name
goto :End
:displayOption
echo Option1: Full Build
echo Option2: Compile Only
echo Option3: Preprocessor Only
goto :takeUserInput
:End
echo End of Batch Script

Using Labels without goto will run the script sequentially (Top-Down)

Basic Functions

Basic Batch Functions

Definig a Function in Batch is similar to Labels but ended with goto :eof

:main
goto :eof

:eof is a predefined label for End of File/Function used to exit a subroutine/Label

To call a Function usecall :Function

Example

@echo off
goto :main

:myFunction
 echo Hello From my Function
goto :eof

:main
  echo Main Function has been called
  call :myFunction
  echo Main Called myFunction
goto :eof

Function Arguments

You can pass a paramter to a Function using

call :myFunction Input1

The function is implemented as

:myFunction
 echo got %~1
goto :eof

%~1 => represents The first input to the funtion


%~2 => represents The Second input to the funtion and so on up to 9 input parameters

Function return

Functions don't have return in Batch but we can mimic the behaviour of function return through variables

Inside a function you can modify any variable outside it

Example

@echo off
goto :main

:myFunction
 echo Hello From my Function
 set %~1=Orange
goto :eof

:main
  echo Main Function has been called
  set myVar=Apple
  call :myFunction myVar
  echo myVar after function call=%myVar%
  echo Main Called myFunction
goto :eof

Function Local Scope

By default, Any function can access any variable but we can set a local scope to a specific function


Preventing any other function from accessing its local variables

By using setlocalat the begining of the function

and usingendlocalat the end of the function which deletes all variables inside that function

Example

@echo off
goto :main

:myFunction
setlocal  echo Hello From my Function
 set localVar=myPrivateValue
  endlocal
goto :eof

:main
  setlocal
  echo Main Function has been called
  set myVar=Apple
  call :myFunction
  echo accessing localVar=%localVar%
  echo Value of localVar can not be accessed!
  echo Main Called myFunction
  endlocal
goto :eof

Script Local Scope

If you use setlocalin all functions inside your script


those variables can not be accessed outside that script
